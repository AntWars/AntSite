Downloads

The best way to download AntWars is cloning our git repo. Here it is: https://github.com/AntWars/AntWars
If you want to get the latest (possible unstable) version you can clone "nightly" branch. Welcome: https://github.com/AntWars/AntWars/tree/nightly
Also you can download archive with one of stable enumerated versions. Please, use links from this table:
